# ContentForge AI 
Multi-agent AI system that automates enterprise content lifecycle:
Product Input → Generate → Compliance → Localize → Distribute
3 hours → 30 seconds | 99% faster

## Features
1. Agent 1: Content generation (LinkedIn posts)
2. Agent 2: Brand compliance checking
3. Agent 3: Hindi localization for India
4. Agent 4: Multi-channel formatting (LinkedIn/X/Email)
5. 99% time reduction

## Setup
1. `pip install -r requirements.txt`
2. create .env: GEMINI_API_KEY=your_key
3. `streamlit run app.py`
