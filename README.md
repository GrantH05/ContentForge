# ContentForge AI

**Enterprise Content Lifecycle Automation — from brief to broadcast in seconds**

Transform raw product input into compliant, multilingual, multi-channel content — in under 30 seconds.

---

## Overview

ContentForge AI is a multi-agent AI system designed to eliminate the bottlenecks in enterprise content creation.

Traditional workflows take 3–6 hours per campaign asset, involving multiple teams across marketing, legal, and localization. ContentForge reduces this to ~30 seconds using an intelligent AI pipeline.

---

## Problem

Enterprise content creation is:

- Slow and resource-intensive
- Dependent on multiple teams (marketing, compliance, localization)
- Difficult to scale across regions and platforms
- Prone to inconsistency in tone and messaging

**Impact:** Delayed campaigns, higher costs, and lost revenue opportunities.

---

## Solution

ContentForge AI automates the entire content lifecycle:

```
Input Brief → Content Generation → Compliance Check → Localization → Distribution
```

No manual handoffs. No delays. Consistent, enterprise-grade output.

---

## Architecture — Multi-Agent Pipeline

| Agent | Function |
|---|---|
| Content Generator | Creates professional, ROI-focused enterprise content using Gemini |
| Compliance Guardian | Ensures brand alignment and safe, professional tone |
| Localization Engine | Translates content into business-ready Hindi (extensible to more languages) |
| Distribution Matrix | Formats output for LinkedIn, X (Twitter), and Email campaigns |

## Setup and Installation

```bash
# Clone the repository
git clone https://github.com/your-username/contentforge-ai.git
cd contentforge-ai

# Install dependencies
pip install -r requirements.txt

# Add your API key — create a .env file:
# GEMINI_API_KEY=your_api_key

# Run the app
streamlit run app.py
```
